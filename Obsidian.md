# **Keyboard Shortcuts**
### **Default Keyboard Shortcuts**
- `ctrl + o` -> Open File
- `ctrl + p` -> Command palette
- `ctrl + n` -> New File
### **My Keyboard Shortcuts**
- `Alt + f` -> Fold All Headings and lists
- `Alt + u` -> Unfold All Headings and lists
