#command 

FTP (**File Transfer Protocol**) is used to transfer files between a client and a server over a network.
```
ftp [OPTIONS] [HOST]
```
HOST can be :
- An IP Address `ftp 192.168.1.10`
- A Domain Name `ftp ftp.example.com`
---
